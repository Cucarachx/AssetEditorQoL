# AssetEditorQoL
Quality of life tools for the Cities: Skylines Asset Editor. Adds subfolder support for the Asset Importer and a Reload Mesh button for the Asset Editor.

Make sure you have all the .dll references correctly added in the solution
	- Assembly-CSharp.dll
   - UnityEngine.dll
   - ICities.dll
   - ColossalManaged.dll
   - CitiesHarmony.Harmony.dll (from Harmony mod folder)
   - CitiesHarmony.API.dll (from Skyve folder)
